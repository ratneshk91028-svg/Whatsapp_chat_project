# WhatsApp Chat Application

A lightweight, backend-driven chat application that demonstrates full CRUD (Create, Read, Update, Delete) capabilities, mimicking basic WhatsApp features.

## ✨ Features
* **View Chats:** Displays all saved messages.
* **New Chat:** Create messages with sender and receiver details.
* **Edit Chat:** Update existing message content.
* **Delete Chat:** Remove messages permanently from the database.

## 🛠️ Tech Stack
* **Runtime:** Node.js
* **Framework:** Express.js
* **Database:** MongoDB & Mongoose ODM
* **Templating:** EJS (Embedded JavaScript)
* **Middleware:** Method-Override (for PUT & DELETE requests)

## 🚀 Setup & Installation

1. Clone this repository:
   ```bash
   git clone <your-repo-link>
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```
3. Start your local MongoDB server.
4. Run the application:
   ```bash
   node index.js
   ```
5. Open your browser and navigate to:
   `http://localhost:8080/chats`
